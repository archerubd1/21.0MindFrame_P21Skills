Activity 1: Rewrite the Message

Turn these rude messages into polite ones using the T.H.I.N.K. rule.

1
RUDE

"You messed it up again."

How can we say this kindly?

Try: "Let's review this part..."
2
DEMANDING

"Send it NOW."

Add some magic words...

Hint: Use "Please" & "Thanks"
3
HURTFUL

"That's a dumb question."

Make it helpful instead...

Try: "Good question! It means..."

Pro Tip: Always start with a positive greeting and end with "Please" or "Thank you". It changes the entire tone!
