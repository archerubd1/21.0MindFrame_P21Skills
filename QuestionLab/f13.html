Build an Inquiry Map

Include these four essential parts in your roadmap

1. The Topic

Choose a broad subject.
(e.g., "Volcanoes")

2. Questions

Ask Why/How questions.
(e.g., "Why do they erupt?")

3. Keywords

List key search terms.
(e.g., magma, lava)

4. Sources

Choose trusted sources.
(e.g., World Book)
