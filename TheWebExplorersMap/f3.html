Browser Basics

Big idea: A browser helps you visit websites

What is a browser?

Your tool for exploring the web

A browser opens web pages.

It helps you go to websites like games, learning, or videos.

It shows text, pictures, and videos.

Websites load inside the browser window.

You use it to visit websites.

You type a web address (URL) or search for a topic.
