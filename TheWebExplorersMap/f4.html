Address Bar

Type URL

This is where you type a website address (URL), like www.example.com.

Check spelling before you press Enter.

Tabs

Many pages

Tabs let you keep more than one web page open at the same time.

Too many tabs? Close the ones you don’t need.

Back / Forward

Go again

Use Back to return, and Forward to go to the next page you visited.

Helpful if you click the wrong link.

Refresh / Reload

Update

Refresh loads the page again. Use it when a page is stuck or not showing right.

Only press it once—then wait a moment.
